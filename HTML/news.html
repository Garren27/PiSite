Post #2 - PiSite

September 1 2019

Today I remembered that I had a Raspberry Pi laying around and I wondered if it would be possible to host a website on it. A quick Google search showed that it was quite possible and many had done so before so there were a few tutorials. After installing a fresh copy of Raspbian on the Pi, installing Apache was easy from the command line. My idea was to host the site on the Pi so I needed a domain name. Unfortunately, the dynamic DNS site the tutorials pointed to were no longer operating, so I had to find a different one. No-ip.com was the site that many recommended, and installation of their Pi side software was easy. This software allowed the Pi to detect if the address that led to my home's network had been changed. The hardest part was setting up the router to allow requests to port 80 in from the Internet since every router is different and some tabs in mine were misleadingly titled. After reviewing some router documentation I had my Pi set up and serving pages to garrenmccallum.ddns.net, which was the most professional looking domain name while being completely free.
